Data Governance

Entiendo el Gobierno de Datos como la base para la confianza en la información. Se trata de establecer procesos para asegurar que los datos sean precisos, consistentes y seguros a lo largo de su ciclo de vida. Mi proyecto de automatización es un ejemplo práctico: valida y estandariza la información de entrada para garantizar la calidad del resultado final.

Python & SQL

Utilizo Python, con librerías como Pandas, para la limpieza y manipulación de datos, y SQL para realizar consultas complejas y extraer información valiosa de bases de datos relacionales. Son las herramientas fundamentales con las que transformo datos crudos en conocimiento accionable.

DataOps

Aplico la mentalidad de DataOps al tratar los flujos de datos como una línea de producción. Mi objetivo es automatizar, monitorear y mejorar continuamente la calidad y la velocidad con la que se entregan los datos. Mi script de automatización es un micro-sistema DataOps: automatiza un pipeline de datos desde la entrada hasta la entrega final.

Process Automation

Mi habilidad principal es identificar tareas manuales y repetitivas y construir soluciones de software para eliminarlas. Con Google Apps Script, pude integrar múltiples aplicaciones (Sheets, Slides, Gmail) para crear un flujo de trabajo autónomo que ahorra tiempo y reduce drásticamente el error humano.

Lean Six Sigma

Utilizo los principios de Lean Six Sigma como un marco estratégico para la resolución de problemas. Me enfoco en definir el problema, medir el estado actual, analizar la causa raíz, implementar mejoras y controlar los resultados. Esta metodología guía mi enfoque tanto en la optimización de procesos como en la mejora de la calidad de los datos.
