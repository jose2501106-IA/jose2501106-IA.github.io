Curiosidad por Datos

Para mí, gobernar datos es tener la curiosidad para prevenir errores, no solo para corregirlos.

Data Governance

Entiendo el Gobierno de Datos como la base para la confianza en la información. Se trata de establecer procesos para asegurar que los datos sean precisos, consistentes y seguros a lo largo de su ciclo de vida. Mi proyecto de automatización es un ejemplo práctico: valida y estandariza la información de entrada para garantizar la calidad del resultado final.

Lean Six Sigma

Utilizo DMAIC como motor de mejora. Me especializo en Diseño de Experimentos (DOE) para optimización de variables críticas ($X$'s), Análisis de Capacidad ($C_pk$) para medir la estabilidad del proceso y Teoría de Restricciones (TOC) para maximizar el *throughput* de información, garantizando eficiencia estadística en los sistemas.

Prompt Engineering

Aplico la Ingeniería del Prompt como una herramienta de Aceleración de Procesos (Lean). Empleo *Zero/Few-Shot Learning* y técnicas de *Chain-of-Thought* para la generación eficiente de código y la exploración rápida de arquitecturas de IA, optimizando la *tasa de transferencia de conocimiento*.
